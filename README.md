# Server Inventory System

## Description
This project is a simple Server Inventory application developed in C#.  
The program allows users to add, display, and delete servers.

Server information is stored in a text file so that the data remains available even after the program is closed.

---

## Technologies
- C#
- .NET Console Application
- Visual Studio
- Git & GitHub

---

## Features
- Add a new server
- Show saved servers
- Delete a server
- Store server data in a file

---


---

## Data Storage
Server data is stored in a file called `servers.txt`.

Example:

WebServer01,192.168.1.10,Linux,Berlin,Running
